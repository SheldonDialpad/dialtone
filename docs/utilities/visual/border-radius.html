Variables

When writing Less, you can set an element's border-radius by using a @border-radius--{type} variable. The table below lists the available variables and its output.

{% for i in borders.radius %} {% assign var = i.class %} {% assign rem = i.rem %} {% endfor %}
Variable Output
@border-radius--{{ var }} {{ rem }}

Classes

Set an element's universal or directional border-radius by using a .d-border-{direction}-radius--{size} class. The table below lists the available classes, border radius size, and the CSS output.

{% assign directions = borders.directions | split: ", " %} {% for dir in directions %} {% if dir == "all" or dir == "top" or dir == "right" or dir == "bottom" or dir == "left" %} {% for i in borders.radius %} {% assign var = i.class %} {% assign rem = i.rem %} {% if dir != "x" or dir != "y" %} {% endif %} {% endfor %} {% endif %} {% endfor %}
Class Output
.d-border{% if dir == "all" %}{% else %}-{{ dir }}{% endif %}-radius--{{ var }} {% if dir == "all" %} border-radius: {{ rem }}; {% elsif dir == "top" or dir == "bottom" %} border-{{ dir }}-left-radius: {{ rem }};
border-{{ dir }}-right-radius: {{ rem }}; {% elsif dir == "right" or dir == "left" %} border-top-{{ dir }}-radius: {{ rem }};
border-bottom-{{ dir }}-radius: {{ rem }}; {% endif %}
