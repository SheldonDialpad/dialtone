--- layout: page title: Padding description: A CSS property which allows you to adjust an element's interior spacing between its child elements and the element's box edge. ---

These utility classes are based on Dialtone's spacing units. Components will typically set their own padding, which is preferred. However components should not create extranenous modifying classes for increasing padding when these utility classes can achieve the same goal. For that reason it is recommended padding modifications should be done using these classes.

All element and component spacing is based on a base-4 pixel base. To apply spacing from Dialtone's Less library, include the desired class name within your semantic class.

{% highlight css %} .semantic-class { .d-padding16; } {% endhighlight %}

This will output:

{% highlight css %} .semantic-class { padding: 1.6rem; } {% endhighlight %}

Adding directional padding

You can add padding to an element by using a universal padding class (i.e. .d-padding4) or by using a directional padding class (i.e. .d-padding-{top|right|bottom|left|y|x}4).
