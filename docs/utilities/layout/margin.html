--- layout: page title: Margins description: A CSS property which allows you to adjust an element's exterior spacing between other objects. ---

These CSS utility classes are based on Dialtone's spacing units. They are used to visually separate elements. However the visual separation required is highly contextual. For that reason, margin is rarely added to a component instead declared only needed within specific layouts or views.

All element and component spacing is based on a base-4 pixel base. To apply spacing from Dialtone's Less library, include the desired class name within your semantic class.

{% highlight css %} .semantic-class { .d-margin16; .d-margin-top8--negative; } {% endhighlight %}

This will output:

{% highlight css %} .semantic-class { margin: 1.6rem; margin-top: -0.8rem; } {% endhighlight %}

Directional Margins

You can add margins to an element by using a universal margin class (i.e. .d-margin4) or by using a directional margin class (i.e. .d-margin-{top|right|bottom|left|y|x}4).

Negative Margins

You can add negative margins to an element to pull it back by using a universal (i.e. .d-margin4--negative) or a directional class (i.e. .d-margin-{top|right|bottom|left|y|x}4--negative).

{% assign dir = spacing.directions | split: ", " %} {% for i in dir %} {% for value in spacing.values %} {% assign val = value.value %} {% assign output = value.output %} {% endfor %} {% endfor %}
Class Output
.d-margin{% if i != "All" %}-{{ i }}{% endif %}{{ val }}--negative {% if i == "y" %} margin-top: -{{ output }};
margin-bottom: -{{ output }}; {% elsif i == "x" %} margin-right: -{{ output }};
margin-left: -{{ output }}; {% else %} margin{% if i != "All" %}-{{ i }}{% endif %}: -{{ output }}; {% endif %}

Auto Margins

Auto margins allow an element to fill a remaining space within an object. This is especially useful in flex layouts.

margin-bottom: auto; {% elsif i == "x" %} margin-right: auto;
margin-left: auto; {% else %} margin{% if i != "All" %}-{{ i }}{% endif %}: auto; {% endif %}
