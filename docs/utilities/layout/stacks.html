--- layout: page title: Stacks & Flows description: Stacks and flows are preset layouts which inject margins between sibling elements within a parent element. This method is favored over applying margins to an element, as it allows for greater modularity and flexibility with layouts. ---

Stacks

The Stack layout works by using the adjacent sibling combinator (+) to apply a top margin only to subsequent elements, meaning it will only work on the second item. Furthermore, to allow for differing nesting Stack values, these margins are scoped to apply only to direct children of the parent (e.g. .d-stack[#] > * + *).

Flows

The Flow layout is similar to the Stacks layout above, except instead of apply a top margin to adjacent children, it applies a left margin. This is useful for when you want to space our elements horizontally.
